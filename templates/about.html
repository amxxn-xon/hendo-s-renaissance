About this dictionary

A {{ meta.get('dict_label','') }} dictionary, made by Hendo Projects with IIT Goa. Every entry was prepared ahead of time from real, named sources and is shown to you exactly as prepared — this site never invents a spelling, a meaning, or a pronunciation. If a word isn't here yet, it says so honestly, and your search helps decide what gets added next.

Ways to use it

  • Search in {{ 'Syriac' if cfg.slug == 'syriac' else 'Arabic' }}, or just spell the word out in English letters or Malayalam — the dictionary matches it to the closest entry.
  • Root tree — see how words grow from the same root, and open any root as an interactive map of its word family.
  • Every occurrence — each entry can show all the places the word appears in {{ meta.get('corpus_label','the text') }}, not just the first.
  • Elsewhere on the web — at the bottom of an entry we pull live results from Wiktionary, the {{ 'Aramaic' if cfg.slug == 'syriac' else 'Arabic' }} Wikipedia{% if cfg.slug == 'arabic' %} and Wikidata{% endif %} for a second opinion. Those are clearly marked as outside sources: helpful, but not checked by us and not part of this dictionary.
