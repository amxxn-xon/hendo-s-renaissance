# Fonts

Drop **EastSyriacAdiabene.otf** (Beth Mardutho Meltho fonts) into this
directory — `style.css` @font-face points at that exact filename. The
Meltho fonts are freeware but are **not redistributed** in this repo;
get them from Beth Mardutho.

Without the file, the stack falls back to *Noto Sans Syriac Eastern*
(requested from Google Fonts in `base.html`) and then to any system
Syriac font. Malayalam uses *Noto Sans Malayalam* with system fallback.
